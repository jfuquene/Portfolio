Welcome to mymovie search app

A place to find happiness

Movie Search App is currently a working progress but here is the basic example

Utilized themoviedb Api to fetch information about movies such as title, rating, release date and a brief description.

Leveraged React's UseState to update the list of movies that come up.

For styling I used Grid to center the movie cards and mediaQueries to make it be responsive.
